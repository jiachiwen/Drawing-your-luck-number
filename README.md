# Drawing-your-luck-number
## A game for customers to get their special coupon

### Guideline

- 消費者輸入自己消費的金額，每達500則可獲得增加隨機抽取數字的機會，舉例：500可抽1數字，2000可抽4數字
- 隨機抽數字
- 3個2，1個0，即可開二獎；若完整抽到2022字樣，則得頭獎
